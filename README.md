# Mobix POC

```
A Node.js 8.0.0+ setup with [yarn](https://yarnpkg.com/) is recommended.
```
## In 'src/services/api' insert the token

## In root directory execute:

```bash
# install dependencies
yarn

# ...or if you'd like to use npm instead
npm install

```

### To start run:

```bash
yarn start

npm run start
```

### To test run:

```bash
yarn test

npm run test
```
